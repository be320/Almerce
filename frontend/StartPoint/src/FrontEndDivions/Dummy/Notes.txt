npm install 

1- To start ths project you need to place this path:
    A- cd frontend/StartPoint
    B- npm install react-scripts --save
    C- npm install 
    D- npm start 

2- Install Redux DevTools from chrome extensions

3- Install mic-recorder-to-mp3 npm package
    npm install --save mic-recorder-to-mp3
  
4- Install Axios library for RestApi
    npm install axios
    Good Documentation source : https://www.npmjs.com/package/axios

5- Install redux-thunk for Async/sync operations
    npm install redux-thunk

6- Install auto-sized textarea
    npm install react-textarea-autosize

7- Install Material Icons
    npm install @material-ui/icons
  
8- Instal Star Rating 
    npm install @material-ui/lab

9- For Image Processing 
    npm install browser-image-compression
    npm install  image-to-base64

10- Loader
    npm install react-loader-spinner --save
